## Installation :

* `apt update`
* `apt install git curl php openssh -y`
* `git clone https://github.com/mrjarvisofficial/Zphisher.git`
* `cd zphisher`
#### Note : Connect your ngrok account with your ngrok token
#### > Run : `bash zphisher.sh`
